Core lessons from existing LanePulse Trainer app:

1. Group selection and group loading must be separate.

   * selected_group_id
   * loaded_group_id
   * next_group_id
     must never be mixed.
2. Do not use group name as primary key. Always use group_id.
3. Timer screen must default to fast poolside operation, not big scattered cards.
4. Fast Timing Console must be the default screen.
5. Finish Capture Mode and Lap Capture Mode are required because swimmers can finish close together.
6. Text visibility is critical. No white text on white background.
7. Buttons must be large and coach-friendly.
8. Session distance must be stored separately.
9. Free Style 100 mtr should be renamed to Free Style.
10. Style and distance must both be used in analysis. Do not compare 50m with 100m.
11. Swimmer history must be preserved even if swimmer is deactivated.
12. Admin delete must be controlled and confirmation-based.

8. Analysis Dashboard

Create intelligent coaching reports.

Important comparison rule:
Compare only same:

* Swimmer
* Style
* Distance

Do not compare:

* 50m Free Style with 100m Free Style
* Back Stroke with Free Style

Reports required:

A. Swimmer Improvement Report
For selected swimmer/style/distance:

* Previous best time
* Latest time
* Difference in seconds
* Improvement %
* Status:
  Improved / Slower / Same / Not enough data
* Chart trend over time

Formula:
Lower time is better.
Improvement Seconds = Previous Best Time - Latest Time.
Improvement % = Improvement Seconds / Previous Best Time * 100.

B. Current vs Previous Timing Report
For each swimmer:

* Last session time
* Previous session time
* Change seconds
* Change %
* Direction:
  Improved / Slower / Same

F. Training Recommendation Engine

Generate practical trainer recommendations based on results.

Rules:

* If swimmer improved:
  Recommend continuing current training with slight progression.
* If swimmer got slower:
  Recommend checking fatigue, technique, starts, turns, breathing rhythm.
* If lap times get slower each lap:
  Recommend endurance and pacing sets.
* If first lap is too fast and later laps drop sharply:
  Recommend controlled start and strong finish drills.
* If lap times are consistent:
  Recommend sprint finish and power work.
* If no enough data:
  Recommend recording more sessions.

Style-based recommendations:
Free Style:

* Streamline push-off drills
* Kick endurance sets
* Breathing rhythm practice
* 25m sprint repeats
* Catch and pull technique drills

Back Stroke:

* Body rotation drills
* Back kick sets
* Straight-arm recovery control
* Wall push-off practice
* Head position stability

Butterfly:

* Dolphin kick drills
* Pull-kick timing
* Short technique repeats
* Core strength sets
* Breathing timing practice

Breast Stroke:

* Kick timing drills
* Glide control
* Pull-breath-kick coordination
* 25m technique repeats
* Turnout kick efficiency

Individual Medley:

* Transition turns
* Stroke balance
* Weak stroke targeted sets
* Pace management

Relay:

* Reaction and changeover practice
* Sprint starts
* Team order optimization

Show recommendation in friendly coach language:

* What happened

* What to train next
